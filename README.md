## Installation
### 1. Clone Repository
```bash
git clone https://github.com/HarithHarizal/Team-14-Code-Visualizer.git
cd Team-14-Code-Visualizer
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Application
```bash
python main.py
```

## How to Use
#### 1. Click "Select File" to select and upload any python file
#### 2. Click "Parse File" after file is uploaded
#### 3. Click "Show Diagram" to view diagram
#### (Optional) Click "Save Image" and enter a name. Select one of the three file formats then select "Save" to save the image to the same directory as the main.py source code.
